# macOS Manual Setup

Settings blocked by SIP or app sandboxing on macOS 26. Cannot be automated via `defaults write`.

## Reduce Transparency

System Settings > Appearance > Reduce transparency

## Safari Developer Menu

Safari > Settings > Advanced > Show Develop menu in menu bar

## Safari WebKit Extras

Enabled automatically when Develop menu is turned on above.

## Mail — Copy Addresses Without Names

Mail > Settings > Composing > When copying addresses, include only the email address

## Spotlight — Exclude Mounted Volumes

System Settings > Spotlight > Privacy > add `/Volumes`

## Messages — Disable Smart Quotes

Messages > Settings > General > uncheck "Smart quotes"

## Messages — Disable Spell Check

Messages > Settings > General > uncheck "Correct spelling automatically"

## Restart on Freeze

System Settings > General > Login Items & Extensions > Allow restart when the computer freezes
